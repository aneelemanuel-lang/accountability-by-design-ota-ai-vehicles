# Accountability-by-Design OTA for AI-Enabled Safety-Critical Vehicle Functions

**University of Surrey | MSc Artificial Intelligence**

---

## Overview

This research proposal investigates how accountability-by-design principles can be integrated into Over-the-Air (OTA) software update architectures for AI-enabled safety-critical vehicle systems.

The project explores AI model provenance, traceability, explainability, evidence logging and incident reconstruction to improve transparency, regulatory compliance and public trust in intelligent transportation systems.

---

## Research Objectives

- Analyse limitations of current OTA architectures.
- Investigate AI model provenance and lifecycle traceability.
- Study AI accountability in safety-critical systems.
- Design an accountability-oriented OTA architecture.
- Develop a conceptual framework for incident reconstruction.
